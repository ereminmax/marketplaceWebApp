Подготовка к выполнению задания.
	1. Запустите SQL Developer. Подключитесь к вашему локальному серверу с помощью своей учетной записи.
	2. Запустите скрипт MPInstall.sql. Он создаст все необходимые объекты и заполнит данные.
	3. В время установки сохраняется лог файл -  MPInstall.log. Проверьте, что бы в нем не было ошибок.
	4. Для "переустановки" запустите сначала скрипт DeleteTables.sql, а затем MPInstall.sql.
